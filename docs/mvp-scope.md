# MVP Scope

## Goal

Build a fast, family-aware second-hand book hunting assistant.

## In Scope

- Shelf photo capture
- Book title recognition from shelf photos
- AI-assisted recommendation
- Duplicate detection across the family library
- Temporary scan results retained for 7 days
- Re-scan flow for purchased books
- Basic family accounts
- Ownership assignment to a family member
- Wishlist support
- Manual recommendation profile setup
- Bilingual UI support for English and Chinese
- Bilingual book metadata display where available

## Primary User Flow

1. Scan a shelf in a second-hand shop.
2. Review the list of detected books.
3. See recommendation strength and duplicate warnings.
4. Decide what to buy.
5. Scan purchased books again at home.
6. Add them to the family library.

## Out of Scope

- Complex permission management
- Social sharing
- Public reviews platform
- Advanced statistics dashboard
- Automatic preference learning as the only recommendation method
- Deep version-market analysis
- Full localization tooling beyond English and Chinese for the first version

## First Release Success Criteria

- The shelf scan flow feels useful within seconds.
- Duplicate warnings are trustworthy enough to influence buying decisions.
- Family library ownership is simple enough to use without friction.
- Users can complete the first setup with a small number of books.
- Book search and detail screens can show both English and Chinese text where available.
